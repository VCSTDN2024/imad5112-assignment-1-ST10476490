2. Project Overview
-------------------



The meal choice is a mobile application developed as part of an assignment in the IMAD5112 subject. This application was created using **Kotlin** and **Android Studio**. The app's primary purpose is to make peoples lives easier by making and app that suggests what to eat during the day

The app was developed to meet the requirements of the assignment, which includes creating a functional mobile app and utilizing GitHub for version control and CI/CD automation using GitHub Actions.
```
